# Crypto Unicorns: Asset System Deep Dive

## ü¶Ñ Unicorn Genetics

Unicorns are sophisticated NFTs with genetic traits stored through efficient bit manipulation:

```solidity
// Simplified Genetics Contract
contract UnicornGenetics {
    // Each Unicorn's genes are stored in a single uint256
    // Format: [Class(8)][Color(8)][Pattern(8)][Horn(8)][Eyes(8)][Special(8)][Stats(64)]
    struct Genes {
        uint256 geneticCode;
    }
    
    function decodeGenes(uint256 geneticCode) public pure returns (
        uint8 class,
        uint8 color,
        uint8 pattern,
        uint8 horn,
        uint8 eyes,
        uint8 special,
        uint64 stats
    ) {
        class = uint8(geneticCode >> 184);
        color = uint8(geneticCode >> 176);
        pattern = uint8(geneticCode >> 168);
        horn = uint8(geneticCode >> 160);
        eyes = uint8(geneticCode >> 152);
        special = uint8(geneticCode >> 144);
        stats = uint64(geneticCode);
    }
}
```

### Classes & Traits
- **8 Primary Classes**: Cloud, Rainbow, Heart, Crystal, Star, Candy, Moon, Flower
- **Genetic Inheritance**: Traits passed down through breeding
- **Stat Distribution**: Encoded in the lower 64 bits
- **Special Traits**: Rare genetic mutations that provide unique abilities
